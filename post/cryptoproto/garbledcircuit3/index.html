混淆电路(三)：GMW协议

Jun 14, 2020 • 2 minutes to read

这篇文章介绍GMW的安全两方计算协议1，并对Yao和GMW的安全两方计算协议进行比较。

GMW协议的使用的基本思想和门运算如下：

参与者$$A,B$$的输入分别为$$a,b$$.

  • $$A$$随机选择两个秘密值$$a_1,a_2$$，使得$$a=a_1 \oplus a_2$$. 保留$$a_1$$，分享$$a_2$$。
  • $$B$$随机选择两个秘密值$$b_1,b_2$$，使得$$b=b_1 \oplus b_2$$. 保留$$b_1$$，分享$$b_2$$。

非交互的XOR门：$$c_1=a_1 \oplus b_1; c_2=a_2 \oplus b_2$$.

交互的AND门：

重组输出：$$d=d_1 \oplus d_2$$

使用R-OT预计算乘法对2

目标：使用2次随机OT(R-OT)生成一个乘法对 $$c_1 \oplus c_2 = (a_1 \oplus a_2)(b_1 \oplus b_2)$$

  • $$P_1$$的输出为：$$\textcolor{blue}{a_1,b_1,c_1}$$
  • $$P_2$$的输出为：$$\textcolor{#EB4892}{a_2,b_2,c_2}$$
  • 特征：$$\textcolor{blue}{c_2} \oplus \textcolor{#EB4892}{c_2} = (a_1 \oplus a_2)(b_1 \oplus b_2) = \textcolor{blue}{a_1 b_1} \oplus \textcolor{#EB4892}{a_2} \textcolor{blue}{b_1} \oplus \textcolor{blue}{a_1} \textcolor{#EB4892}{b_2} \oplus \textcolor{#EB4892}{a_2 b_2}$$

协议内容如下：

  1. $$P_2$$选择$$\textcolor{#EB4892}{a_2} \in_R \lbrace 0,1 \rbrace $$
  2. $$P_1$$和$$P_2$$执行随机OT：$$P_1$$(发送者)获得$$(m_0,m_1)$$, $$P_2$$(接收者)输入$$\textcolor{#EB4892}{a_2}$$并获得$$u_2=m_{a_2}$$
  3. $$P_1$$设置$$\textcolor{blue}{b_1}=m_0 \oplus m_1$$及$$v_1=m_0$$
    $$\circ$$ 观察 $$v_1 \oplus u_2 = m_0 \oplus m_{a_2}$$
    $$\circ$$ 如果 $$a_2=0$$则$$v_1 \oplus u_2 = m_0 \oplus m_0=0=\textcolor{#eb4892}{a_2} \textcolor{blue}{b_1}$$
    $$\circ$$ 如果 $$a_2=1$$则$$v_1 \oplus u_2 = m_0 \oplus m_1=b_1=\textcolor{#eb4892}{a_2} \textcolor{blue}{b_1}$$
  4. $$P_1$$和$$P_2$$交换角色，重复步骤1-3获得$$(a_1,u_1)$$和$$(b_2,v_2)$$
  5. $$P_i$$设置$$c_i=(a_ib_i) \oplus u_i \oplus v_i$$

正确性：$$\textcolor{blue}{c_1} \textcolor{#eb4892}{c_2}=\textcolor{blue}{a_1b_1} \oplus \textcolor{blue}{u_1} \oplus \textcolor{blue}{v_1} \oplus \textcolor{#eb4892}{a_2b_2} \oplus \textcolor{#eb4892}{u_2} \oplus \textcolor{#eb4892}{v_2} = \textcolor{blue}{a_1 b_1} \oplus \textcolor{#EB4892}{a_2} \textcolor{blue}{b_1} \oplus \textcolor{blue}{a_1} \textcolor{#EB4892}{b_2} \oplus \textcolor{#EB4892}{a_2 b_2} $$
